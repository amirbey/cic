# Platform as a Service (PaaS)


<p><span style="font-size:14px">Platform as a Service (PaaS)&nbsp;refers to cloud computing services that supply an on-demand environment for developing, testing, delivering, and managing software applications. PaaS is designed to make it easier for developers to quickly create applications (e.g., web, mobile apps), without worrying about setting up or managing the underlying infrastructure of servers, storage, network, and databases needed for development.</span></p>

<p><img src="https://hallways.cap.gsa.gov/system/files/PaaS-1555525170.png" /></p>

<p><span style="font-size:14px">Platform as a service (PaaS) is a complete development and deployment environment in the cloud, with resources that enable you to deliver everything from simple cloud-based apps to sophisticated, cloud-enabled enterprise applications. You purchase the resources you need from a Cloud Service Provider (CSP)&nbsp;on a pay-as-you-go (i.e., utilization)&nbsp;basis and access them over a secure Internet connection.</span></p>

<p>&nbsp;</p>

<p><span style="font-size:14px">Like IaaS, PaaS includes infrastructure&mdash;servers, storage, and networking&mdash;but also middleware, development tools, Business Intelligence (BI) services, database management systems, and more. PaaS is designed to support the complete&nbsp;application lifecycle: building, testing, deploying, managing, and updating.</span></p>

<p>&nbsp;</p>

<p><span style="font-size:14px">PaaS allows you to avoid the capital expenses (CapEx)&nbsp;and complexity of buying and managing software licenses, the underlying application infrastructure and middleware or the development tools and other resources. You manage the applications and services you develop, and the cloud service provider typically manages everything else.</span></p>

<p>&nbsp;</p>

<p><span style="font-size:18px"><span style="color:#003366"><strong>PaaS Supports</strong></span></span></p>

<ul>
	<li><span style="font-size:16px"><strong>Development framework</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">PaaS provides a framework that developers can build upon to develop or customize cloud-based applications. Similar to the way you create a Microsoft&nbsp;Office Excel macro, PaaS lets developers create applications using built-in software components. Cloud features such as scalability, high-availability, and multi-tenant capability are included, reducing the amount of coding that developers must do.&nbsp;</span></p>

<ul>
	<li><span style="font-size:16px"><strong>Analytics or business intelligence</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">Tools provided as a service with PaaS allow organizations to analyze and mine their data, finding insights and patterns and predicting outcomes to improve forecasting, product design decisions, investment returns, and other business decisions.&nbsp;</span></p>

<ul>
	<li><span style="font-size:16px"><strong>Additional services</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">PaaS providers may offer other services that enhance applications, such as workflow, directory, security, and scheduling.&nbsp;</span><br />
&nbsp;</p>

<p><span style="color:#003366"><span style="font-size:18px"><strong>PaaS Advantages</strong></span></span></p>

<p><span style="font-size:14px">By delivering infrastructure as a service, PaaS offers the same advantages as IaaS. But its additional features&mdash;middleware, development tools, and other business tools&mdash;give you more advantages:&nbsp;</span><br />
&nbsp;</p>

<ul>
	<li><span style="font-size:16px"><strong>Cut coding time</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">PaaS development tools can cut the time it takes to code new applications with pre-coded application components built into the platform, such as software libraries, workflow, directory services, security features, search, and so on.&nbsp;</span></p>

<ul>
	<li><span style="font-size:16px"><strong>Add development capabilities without adding staff</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">PaaS&nbsp;components can give your development team new capabilities without you needing to add staff having the required skills.&nbsp;</span></p>

<ul>
	<li><span style="font-size:16px"><strong>Develop for multiple platforms&mdash;including mobile&mdash;more easily</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">Some service providers give you development options supporting&nbsp;multiple platforms, such as computers, mobile devices, and browsers making cross-platform applications development quicker and easier.&nbsp;</span></p>

<ul>
	<li><span style="font-size:16px"><strong>Use sophisticated tools affordably</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">A pay-as-you-go (e.g., utilization) model makes it possible for individuals or organizations to use sophisticated development software, Business Intelligence, and analytics tools that would you would not otherwise be able to access or afford to purchase.&nbsp;</span></p>

<ul>
	<li><span style="font-size:16px"><strong>Support geographically distributed development teams</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">Because the development environment is accessed over the Internet, development teams can work together on projects even when team members are in remote locations.&nbsp;</span></p>

<ul>
	<li><span style="font-size:16px"><strong>Efficiently manage the application lifecycle</strong></span></li>
</ul>

<p style="margin-left:40px"><span style="font-size:14px">PaaS provides all of the capabilities that you need to support the complete application lifecycle: building, testing, deploying, managing, and updating within the same integrated environment.</span></p>

<p>&nbsp;</p>
